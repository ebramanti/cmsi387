    • cd

    cd is the notation for changing directories in Unix.

    Example: 

     Edwards-MacBook-Air:CMSI Edward$ cd cmsi387/

    In the above example, cd will direct the user from the current folder, CMSI, to the directory cmsi387/. If cd is entered without any arguments, it will take the user to the home directory.

    ls

    ls lists the current directory's contents.

    Example: 

    Edwards-MacBook-Air:cmsi387 Edward$ ls
README.md homework

    The example shows the current directory is cmsi387. When ls is entered into the command line, it shows all the current files and subfolders located within the directory.

    mkdir

    mkdir makes a directory (or subfolder) within the current directory.

    Example: 

    Edwards-MacBook-Air:cmsi387 Edward$ mkdir assignments
Edwards-MacBook-Air:cmsi387 Edward$ ls
README.md assignments homework

    The example shows the current directory is cmsi387. When mkdir assignments is inputted into the command line, it creates a subfolder in the current directory (NOTE: mkdir does not output confirmation of directory creation). Therefore, using the command ls that we used above, we discover that indeed mkdir created a new subfolder in the cmsi387 directory (in this case, assignments).

    rm

    rm removes files or directories from the current directory.

    Example: 

    Edwards-MacBook-Air:cmsi387 Edward$ ls
README.md assignments homework  junk.txt
Edwards-MacBook-Air:cmsi387 Edward$ rm junk.txt
Edwards-MacBook-Air:cmsi387 Edward$ ls
README.md assignments homework

    In this example, rm is called on junk.txt, which subsequently deletes the file without any output confirmation. ls shows that the file is indeed deleted when the command is invoked like so (NOTE: rm does not work when applied to folders unless invoked like so: rm -rf directory).

    date

    date prints/sets the system time/date.

    Example: 

    Edwards-MacBook-Air:cmsi387 Edward$ date
Wed Jan 1 12:00:01 PST 2014

    When date is entered into the shell above, it returns the current date, time, and time zone in the shell. To modify or change the date, invoke date 042708002010 (this will change computer date to April 27, 2010 8:00am).

    ssh

    ssh utilizes the OpenSSH client to allow for the user to login to a remote machine and execute commands.

    Example: 

    Edwards-MacBook-Air:cmsi387 Edward$ ssh user@my.cs.lmu.edu
user@my.cs.lmu.edu's password:

    ssh requires two entries: a valid username that has the correct permissions and the hostname of the remote machine (in this case, user and my.cs.lmu.edu). If entry is successful, the terminal will prompt for a password, and connect you to the remote host after entry.

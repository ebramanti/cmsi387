HOME

HOME is the variable notation for the home directory of the user.

LANG

LANG is a variable that determines the locale category for native language, which can be used to determine the language to use for error messages, instructions, etc.

PATH

PATHis an environment variable specifying a set of directories where executable programs are located.

/dev/null

/dev/null is an infinite data source/sink; data written to /dev/null is discarded, and reads from /dev/null shall always return end-of-file.

&

& runs and executes the command and gives you the prompt back right away (works in background).

~

~ represents several different pathnames:
  • Home directory
  • Current, or previous, working directory
  • Directory from directory stack

    • ||

    || is the logical OR operator. If the exit status of the first command is not 0, then execute the second command.

    *

    * is a symbol that allows string matching of any characters, including null.
